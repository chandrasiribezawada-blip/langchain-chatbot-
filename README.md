# langchain-chatbot-
A simple LangChain-based chatbot

## Setup Instructions

### 1. Install dependencies
pip install -r requirements.txt

### 2. Add your OpenAI API key
Create a .env file:
OPENAI_API_KEY=your_api_key

### 3. Run the app
streamlit run app.py

## Features
- Chatbot using LangChain
- Conversation memory
- Streamlit UI
